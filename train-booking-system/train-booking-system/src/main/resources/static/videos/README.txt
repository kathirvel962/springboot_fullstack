Place your video file here and name it `fast_train.mp4`.

Recommended format and encoding:
- Container: MP4 (.mp4)
- Video codec: H.264 (AVC)
- Audio codec: AAC
- Resolution: 1280x720 or 1920x1080
- Bitrate: 2-8 Mbps depending on resolution

Example path (when project runs): /videos/fast_train.mp4

If you want to use a different filename, update the <source> element in `templates/trains/list.html` accordingly.
